roles/run.admin
Allows you to create, update, delete, and configure Cloud Run services.

roles/iam.serviceAccountAdmin
Allows you to create and manage service accounts.

roles/iam.serviceAccountUser
Allows you to attach service accounts to Cloud Run services. This fixes the iam.serviceAccounts.actAs error.

roles/resourcemanager.projectIamAdmin
Allows you to grant project-level IAM roles to service accounts and users.

roles/serviceusage.serviceUsageAdmin
Allows you to enable required APIs, like Cloud Run, Cloud Build, Vertex AI, BigQuery, Storage, IAM Credentials.

roles/artifactregistry.admin
Allows you to manage Artifact Registry repositories/images used by Cloud Run deployments.

roles/cloudbuild.builds.editor
Allows you to run/manage Cloud Build builds.
